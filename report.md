# Cloud Cost Comparison: AWS vs Azure

## 📌 Project Overview
This project explores the intersection of cloud architecture and financial strategy.  
Learners will conduct a **comprehensive cost comparison** between **Amazon Web Services (AWS)** and **Microsoft Azure** for hosting a small web application.  

The goal is to provide practical insights into:
- Cloud economics
- Pay-as-you-go pricing models
- Key cost drivers (compute, storage, egress)
- Long-term sustainability of cloud choices

---

## 🎯 Project Goals
- Understand the core pricing models of AWS and Azure (per-second/per-minute billing, free tiers).
- Identify and compare equivalent services (e.g., EC2 vs. Azure Virtual Machines, S3 vs. Blob Storage).
- Analyze the impact of licensing (Azure Hybrid Benefit) and architectural choices (multi-zone data transfer).
- Learn to use cloud pricing calculators for accurate financial estimates.
- Develop cost-optimization strategies (instance right-sizing, commitment-based discounts).
